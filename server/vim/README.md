Vim конфиг для сервера
==============

.vimrc, simple configures for server, without plugins.

# Introduction

This repository is a simplified version of [k-vim](https://github.com/wklken/k-vim)

Just remove all plugins, keep basic config.

# Install

#### 1. backup your old .vimrc if it is necessary

```
cp ~/.vimrc ~/.vimrc_bak
```

#### 2. just get the file

```
curl https://raw.githubusercontent.com/vaeum/cdn/master/server/vim/vimrc > ~/.vimrc
```
